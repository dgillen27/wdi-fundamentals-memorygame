This game is the greatest game of all time! I hope you have some fun playing it.

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-sujaku, Pexeso, or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards flipped face up over each turn. The object of the game is to turn over pairs of matching cards.
